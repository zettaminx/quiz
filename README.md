# 🔐 정보보안기사 Master 퀴즈 앱

28개년 정보보안기사 실기 단답형 기출문제를 완벽하게 학습할 수 있는 퀴즈 앱입니다.

## 🚀 사용 방법

### 1. 웹에서 바로 사용 (추천)
아래 링크를 클릭하면 설치 없이 바로 퀴즈를 풀 수 있습니다. 스마트폰에서 '홈 화면에 추가'를 하면 앱처럼 사용할 수 있습니다.
👉 **[퀴즈 앱 시작하기](https://zettaminx.github.io/quiz/)**

### 2. PC용 실행 파일(.exe) 다운로드
1. 상단 메뉴의 **[Releases](https://github.com/zettaminx/quiz/releases)**를 클릭합니다. (현재 준비 중이라면 아래 '소스 코드' 방법을 이용하세요)
2. 가장 최신 버전에서 `SecurityQuiz_Ultimate_v3.exe` 파일을 다운로드하여 실행합니다.

### 3. 소스 코드 다운로드
1. 오른쪽 상단의 초록색 **[<> Code]** 버튼을 누릅니다.
2. **[Download ZIP]**을 클릭하여 압축 파일을 내려받습니다.

---

## 📱 주요 기능
- 500개 이상의 최신 기출 데이터 연동
- 과목별 카테고리 분류 및 회차별 정보 제공
- 정답 및 상세 해설(Commentary) 포함
- 오프라인 모드 및 PWA 설치 지원
